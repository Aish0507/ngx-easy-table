Table options:

Name Description Type
searchEnabled Enable search for each column boolean
headerEnabled Enable header visibility for each column boolean
orderEnabled Enable order option for each column boolean
orderEventOnly (optional) You can define your own sort action, for example by Lastname if you display First and Last name. Click here for the details. boolean
paginationEnabled Enable pagination visibility boolean
clickEvent Enable logger. Each action is traced in the console. boolean
selectRow You can highlight row by clicking boolean
selectCol You can highlight column by clicking boolean
selectCell You can highlight cell by clicking boolean
rows Set how many table rows is visible when table is rendered number
additionalActions (experimental) For now it is just placeholder, that add additional column on the right side of the table. Additional features there will be added in the next releases (started from 10.0) bool
serverPagination By enabling this option normal pagination is turned off. boolean
isLoading Set table in loading state. boolean
detailsTemplate By clicking arrow on the right side of the table additional tow with details appears. boolean
groupRows Enable row grouping. See https://ngx-easy-table.eu/#/group-rows for more details. boolean
paginationRangeEnabled Show/ hide pagination range boolean
collapseAllRows Display all rows details at once. boolean
checkboxes Show additional column with checkboxes boolean
resizeColumn Column resizer appears on the right side of each column. boolean
fixedColumnWidth By setting to tru table calculates best (fixed) width for each column. boolean
horizontalScroll Enable horizontal scroll. For example, when you have many columns. boolean
draggable Option to drag/drop rows. (only rows now supported) boolean
logger Enable logger. Each action is traced in the console. boolean
showDetailsArrow (optional) Hide details arrow boolean
showContextMenu (optional)(experimental) Show context menu instead browser one. boolean
persistState (optional)(experimental) Save the current state once you leave tab with tables. boolean
paginationMaxSize (optional) Maximal amount of the items in the pagination. boolean
tableLayout.style Table row theme. STYLE | string | null ('tiny', 'big', 'normal')
tableLayout.style Table mode. STYLE | string | null ('light', 'dark')
tableLayout.borderless Table with borders or without. boolean | null
tableLayout.hover Hover over table row. boolean | null
tableLayout.striped Every nth row has slightly darker background boolean | null

Column options:

Name Description Example
key Object key.
You can enter here nested object key separated by dot.
title Column title displayed in the table.
placeholder Search placeholder name.
width (optional) Column width in %. When not set table calculate best width.
cellTemplate (optional) You can define your own column template
orderEnabled (optional) You can define if column order is enabled.
searchEnabled (optional) You can define if column search input is enabled.
orderBy (optional) By enabling orderBy option table set default order by this column.
